    • Introduction

    CSS is a language that describes the style of an HTML document.

    CSS describes how HTML elements should be displayed.

    Backgrounds

    You can use CSS to color the backgrounds of your pages, sections, fonts, and more.

    You can also use images as backgrounds.

    Here are some example CSS background properties:

    • background-color
    • background-image
    • background-repeat
    • background-attachment
    • background-position
    Text

    There are many properties you can change on displayed text. You can change the color, font type and size.

    In CSS, there are two types of font family names:

    • generic family - a group of font families with a similar look (like "Serif" or "Monospace")
    • font family - a specific font family (like "Times New Roman" or "Arial")

    Here are some example CSS text properties:

    • color
    • text-align
    • text-transform
    • text-shadow
    • font-size
    • font-family

    Links

    In addition, links can be styled differently depending on what state they are in. The four links states are:

    • a:link - a normal, unvisited link
    • a:visited - a link the user has visited
    • a:hover - a link when the user mouses over it
    • a:active - a link the moment it is clicked

      • /* unvisited link */ a:link { color: red; }
      /* visited link */ a:visited { color: green; }
      /* mouse over link */ a:hover { color: hotpink; }
      /* selected link */ a:active { color: blue; }
